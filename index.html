my work

Image Coregistration

Developed this image correlating program to help validate that the in vivo head and neck tumor image matches the ex vivo gold standard image. This uses both non-rigid and rigid transformations to correlate the images and then plots the average intensities by regions to show that the in vivo image properly detects the tumor.

Language: Matlab

Tiny Search Engine

Developed this search engine used to search through pages of the Dartmouth Computer Science website. This was divided into three sections, the web crawler, indexer, and query. The crawler, given a web URL, crawls through to find all other weboages linked to this. Then the indexer creates an inverted index of all the words and frequencies found. Finally, the query allows users to search for words through the commandline and prints out the webpages containing the words, ranked by frequency.

Language: C

Mission Incomputable

Mission Incomputable is a game containing three components: a field agent, a guide agent, and a game server. The field agent and guide agent communicate through the server to look for code drops that are scattered around a certain location. While trying to neutralize the code drops within a set amount of time, the field agents must also avoid being captured by opposing players. The field agent uses the pebble watch to search for and neutralize code drops while the guide agent uses a computer to send hints.

Language: C

Kevin Bacon Game

The Kevin Bacon game takes a data file containing a list of actors and their related movies to find an actor's "Kevin Bacon" number. This program uses breadth-first search to calculate the number of degrees an actor is separated from Kevin Bacon, being connected through other actors that have been in the same movies until they reach a movie that Kevin Bacon has starred in.

Language: Java
